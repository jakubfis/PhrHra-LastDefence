# Last Defence (v1.4)

https://youtu.be/rP6xYuiOQGE

Last Defence je jednoduchá tower defence hra v neonovém stylu. Vytvořil jsem ji jako IT školní projekt. Úkolem hráče je stavět vojáky a věže tak, aby se nepřátelé nedostali přes mapu.

## Jak hru spustit
1. Stáhni si celou složku s projektem.
2. Otevři soubor [`index.html`](index.html) v jakémkoliv prohlížeči (Chrome, Firefox atd.).
3. Doporučuju mít zapnutý zvuk, bez něj to není ono.

## Dokumentace k projektu
Dokumentaci jsem rozdělil do několika částí, aby se v tom dalo vyznat:

2. [**Design a mechaniky**](docs/02_design.md) - Jak fungují peníze, vojáci a nepřátelé.
3. [**Technické řešení a grafika**](docs/03_technika.md) - Jak se to vykresluje a jak se to ovládá.
4. [**Audio a odezva**](docs/04_audio.md) - Všechno o zvucích ve hře.
